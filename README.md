# Energy World - E-Commerce Platform

**Energy World** is a professional e-commerce website for an electrical and electronics retail shop in Una, Gujarat. Built with HTML5, CSS3, JavaScript, and modern design principles featuring AJAX navigation, responsive design, smooth animations, and glassmorphism effects.

## âœ¨ Key Features

- **Responsive Design** - Mobile-first layout optimized for all devices (< 576px to 2000px+)
- **AJAX Navigation** - Smooth page transitions without full reloads
- **Modern UI** - Glassmorphism effects, gradient overlays, and scroll-triggered animations
- **Form Validation** - Client-side validation with error handling
- **Developer Modal** - Contact information and social media links (WhatsApp, Instagram, Facebook, GitHub)
- **Google Maps Integration** - Location embed on contact page
- **SEO Optimized** - Semantic HTML, meta tags, and structured data
- **High Performance** - Lazy loading, CSS transforms, and optimized code
